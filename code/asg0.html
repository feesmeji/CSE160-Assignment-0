CSE 160 Assignment 0: Vector Library

Author: Fernando Mejia

Description: This program supports most linear algebra vector operations.

Instructions: Enter numerical values for each vectors' x and y values. For simple drawings of vectors, simply press the first "draw" button after entering the x and y values.

To do more complicated vector operations, enter the vectors' x and y values, choose the operation you would like to execute from the drop-down menu, enter a scalar value, and finally, click the draw button.

Please use a browser that supports "canvas"
